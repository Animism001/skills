# 罗伯特·麦基外部观点调研

## 一、外部观察到的模式

### 1. 学术背景与影响力模式
- **信息来源**: http://taggedwiki.zubiaga.org/new_content/fb818b0a4a490261f23a1daa68a0b2a2
- **可信度**: 高（维基百科镜像内容）
- **内容**: 麦基1941年出生，曾在南加州大学任教授，1983年开始开设"STORY Seminar"课程。自1984年以来已有超过50,000学生参加过他的课程。他的学生获得过35次奥斯卡奖、165次奥斯卡提名、170次艾美奖、500次艾美奖提名。

### 2. 教学模式特点
- **信息来源**: https://ensemblewriter.com/docs/story-review
- **可信度**: 中（专业作家网站）
- **内容**: 麦基的方法强调"原则"而非"公式"。他反对模板化的编剧教学，注重讲述故事的底层原理。他的写作风格权威甚至带有说教意味，《故事》一书内容极为密集和学术化。

### 3. 理论渊源模式
- **信息来源**: https://taggedwiki.zubiaga.org/new_content/fb818b0a4a490261f23a1daa68a0b2a2
- **可信度**: 高（维基百科镜像内容）
- **内容**: 麦基讨论的许多观点自亚里士多德以来就一直存在，他将古典叙事理论与现代电影实践相结合。

### 4. 商业成功模式
- **信息来源**: https://book.douban.com/subject/26717137/
- **可信度**: 中（豆瓣读书）
- **内容**: 《故事》一书被称为"编剧圣经"，精装版2014年上市后3天即加印，3个月后成为当当网2014年艺术榜榜首、亚马逊编辑2014年度收官之荐。被英国卫报称为"亚里士多德后最有影响力的讲故事理论家"。

## 二、批评与争议

### 1. 缺乏实际制片经验的批评
- **信息来源**: http://taggedwiki.zubiaga.org/new_content/fb818b0a4a490261f23a1daa68a0b2a2
- **可信度**: 高（维基百科镜像内容）
- **内容**: 编剧乔·埃斯特哈斯（Joe Eszterhas）批评麦基教授编剧却从未有过自己的剧本被拍成电影。麦基回应称："世界上充满了那些教授自己无法做到的事情的人"，并承认虽然他卖掉了所有写的剧本，但由于只是被选项而不是被制作，因此缺乏银幕署名。

### 2. 写作风格与内容密度的批评
- **信息来源**: https://ensemblewriter.com/docs/story-review
- **可信度**: 中（专业作家网站）
- **内容**: 最常见的批评是《故事》对于初学者来说过于密集和学术化。麦基的写作风格权威甚至近乎武断，概念的数量之多可能令人不知所措。新作家常常在读过这本书后感到比开始时更加困惑。

### 3. 反公式立场与说教语气之间的矛盾
- **信息来源**: https://ensemblewriter.com/docs/story-review
- **可信度**: 中（专业作家网站）
- **内容**: 麦基的反公式立场与其写作的说教语气之间存在差距。他坚称没有规则，但随后发表的陈述听起来非常像规则，例如"一个故事必须建立到一个不可逆转变化的高潮"、" exposition必须被戏剧化"。

### 4. 对经典作品的争议性评价
- **信息来源**: https://www.bartleby.com/essay/The-Movie-Was-Downright-Boring-And-Mckee-F3CTNG5YLJ5W
- **可信度**: 低（学生论文网站）
- **内容**: 麦基曾批评《公民凯恩》"没有心"，认为奥逊·威尔斯缺乏风格、内容、情节和情感冲击力。他认为《公民凯恩》的风格和内容是分离的，风格压倒了内容。

## 三、与同行对比

### 1. 与Syd Field（西德·菲尔德）的对比
- **信息来源**: https://www.narratologyjournal.ir/article_147143_57415b14b07d989e1fa71c62eee304e7.pdf
- **可信度**: 中（学术期刊）
- **内容**: 
  - Syd Field认为编剧在开始写作之前必须知道主题
  - Robert McKee反对使用"主题"一词，而是使用"观察思想"的概念
  - Field更强调三幕结构的具体页数位置，而McKee更注重原则而非具体规则

### 2. 与John Truby（约翰·特鲁比）的对比
- **信息来源**: https://www.narratologyjournal.ir/article_147143_57415b14b07d989e1fa71c62eee304e7.pdf
- **可信度**: 中（学术期刊）
- **内容**:
  - Truby引入了"22个关键步骤"和"角色网络"概念
  - 与McKee不同，Truby的方法不是僵化的三幕结构，而是可以根据叙事需要重新排列的22个故事节拍
  - Truby更强调角色驱动的故事，情节从角色欲望和道德选择中有机地产生
  - McKee和Truby都认识到故事不仅是事件，而是"论证"，是竞争价值之间的道德、心理或主题斗争

### 3. 与Christopher Vogler（克里斯托弗·沃格勒）的对比
- **信息来源**: https://www.narratologyjournal.ir/article_147143_57415b14b07d989e1fa71c62eee304e7.pdf
- **可信度**: 中（学术期刊）
- **内容**:
  - Vogler认为"平凡世界"阶段是剧本中主要主题或思想的最佳位置
  - Vogler的方法基于约瑟夫·坎贝尔的英雄旅程神话结构
  - 相比之下，McKee的框架更广泛，不局限于神话模式

### 4. 与Blake Snyder（布莱克·斯奈德）的对比
- **信息来源**: https://peekatthis.com/the-best-screenwriting-books-on-amazon/
- **可信度**: 中（独立影评人网站）
- **内容**:
  - Blake Snyder的《Save the Cat!》提供了更具体的节拍表和类型模板
  - McKee的《故事》更抽象和理论化，不提供具体的页数目标或节拍表
  - Snyder的方法更适合初学者和商业类型片写作，而McKee适合更深入的叙事研究

### 5. 在叙事理论谱系中的位置
- **信息来源**: https://jamespbarkerwriter.com/narrative-formula-toward-conscious-design/
- **可信度**: 中（作家博客）
- **内容**: McKee和Truby被认为是最接近更深层次叙事理解的"古典理论家"。McKee对"期望与结果之间的差距"的强调接近认知失调的概念，但他没有将其识别为神经化学或体验现象。与Dramatica理论相比，McKee的框架仍然专注于外部因果关系而非内部转变。
