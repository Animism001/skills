# 《神雕侠侣》2D动画电影 · 服化道设计方案

> **项目分类**：AI·动画(2D)·电影
> **视觉基调**：国风水墨为底色，参考《大鱼海棠》《白蛇：缘起》
> **原著版本**：金庸《神雕侠侣》三联版
> **时代背景**：南宋末年
> **设计原则**：宋制汉服为骨、水墨意境为魂、角色性格为脉

## 一、杨过

### 1.1 少年期（桃花岛→全真教→古墓入门）

**服装描述**

| 维度 | 设计 |
|------|------|
| 款式 | 交领短襦 + 及膝裤 + 绑腿，南宋少年常服形制。短襦袖口收窄便于活动，衣身偏短显少年利落 |
| 面料 | 粗麻为主，局部棉布。质感偏粗砺，暗示流离身世 |
| 色彩 | 主色灰青（#6B7B8D），内衬月白（#F0EDE5），裤装赭石（#A0785A）。整体灰蒙蒙的，如未干的水墨底稿 |
| 层次 | 单层短襦，无外袍。腰系草编绳带，绑腿用粗布条。层次极简，突出少年的单薄与漂泊感 |

**发型与妆容**

- 发型：束发成总角，头顶双髻，余发披散及肩。发色乌黑但略显毛躁，不似世家子弟的整齐
- 妆容：眉形自然略粗，肤色偏黄带风霜感，左颊有一道浅疤（桃花岛被郭芙所伤），唇色偏淡

**关键配饰/道具**

- 草编腰绳：母亲穆念慈遗物，绳头系一枚磨平的铜钱
- 破旧布包：随身包袱，补丁叠补丁
- 全真教木牌：刻有"全真"二字的入门腰牌，后被杨过掷弃

**武器设计**

- 无专属武器。偶尔拾取树枝石块，战斗风格为徒手搏击与身法闪避，动作设计偏野性、不拘一格

### 1.2 青年期（古墓出关→断臂前）

**服装描述**

| 维度 | 设计 |
|------|------|
| 款式 | 交领长衫 + 广袖外袍 + 束腰。长衫及踝，外袍过膝，袖口渐宽呈垂弧。南宋文人衫与侠客袍的融合形制 |
| 面料 | 外袍用细绢，内衫用素绫。质感从少年期的粗麻跃升，暗示古墓中十六年的清雅生活 |
| 色彩 | 主色月白（#F0EDE5），外袍淡墨青（#8BA5B5），腰封玄色（#2C2C2C）。月白为底、墨青为面，如宣纸上洇开的淡墨。行走时衣袂翻飞，有"白衣胜雪"之意 |
| 层次 | 三层：素绫内衫→细绢长衫→广袖外袍。腰封束于长衫外、外袍内，形成收放对比。外袍可滑落单肩，露出内衫，增加动态层次 |

**发型与妆容**

- 发型：长发及腰，以玉簪松松束于顶，余发自然披散。额前碎发随风，不刻意打理。玉簪为小龙女所赠
- 妆容：眉若远山，眉尾微扬带英气。肤色白皙（古墓长居所致），唇色淡粉。面容清俊，眼神明亮而带三分桀骜

**关键配饰/道具**

- 白玉簪：小龙女赠予，素面无纹，温润如脂
- 丝绦腰带：小龙女亲手编织，银白色，系于腰封之外
- 古墓寒玉床碎片：随身携带的小块寒玉，触之冰凉

**武器设计**

- **玄铁重剑**（此期尚未获得）：暂用全真剑法与古墓剑法，以普通长剑为器。剑身素白无纹，剑柄缠白绫，与白衣相呼应

### 1.3 神雕大侠期（十六年后）

**服装描述**

| 维度 | 设计 |
|------|------|
| 款式 | 右衽半臂 + 宽袖长袍 + 披风。半臂露右臂（断臂侧），左袖完整宽垂。披风及地，行走时如墨云翻涌。整体形制打破对称，以不对称剪裁映射断臂之殇 |
| 面料 | 外袍与披风用重磅绡，半臂用粗绢。质感介于少年粗砺与青年细腻之间，是沧桑与内敛的融合 |
| 色彩 | 主色玄黑（#1A1A2E），内衬暗红（#8B2500），披风墨色渐变（#2C2C2C→#0D0D0D）。玄黑如夜，暗红如血，墨色如渊。整体色调极深，与少年灰青、青年月白形成三段式色彩叙事 |
| 层次 | 三层：暗红内衫→玄黑半臂长袍→墨色披风。右肩披风滑落，露出断臂处空荡的袖管（袖管以暗红丝线缝合封口，如伤口结痂）。左袖宽大垂坠，与空荡右袖形成强烈视觉对比 |

**发型与妆容**

- 发型：长发散披，仅以一根黑色丝带松束于脑后。发间杂有几缕早白（三十六岁已见霜丝），额前碎发凌乱。不束不簪，是放逐江湖的随性
- 妆容：眉间有竖纹（常年蹙眉所致），眼角微垂带沧桑，唇色偏暗。面容从清俊转为刚毅，下颌线条更硬朗。肤色从白皙转为微麦色（十六年风霜）

**关键配饰/道具**

- 面具：黄杨木雕半脸面具，遮住右半脸。面具纹路古朴，眼孔处镂空，戴上面具时仅露左眼与下半张脸。侠客聚义时佩戴，取"不欲以真面目示人"之意
- 断臂袖管：右袖空荡，以暗红丝线缝合封口，缝线走势如经脉断处，是角色最核心的视觉符号
- 神雕翎羽：披风肩部缝有三片巨雕翎羽，墨黑泛紫光，是神雕所赠

**武器设计**

- **玄铁重剑**：剑身通体玄黑，无锋无刃，宽厚如碑。剑格处铸有古朴云纹，剑柄以玄铁丝缠绕。重约七十余斤，杨过单手持握。剑身无光却沉凝如山，与"重剑无锋、大巧不工"的武学境界呼应。挥动时空气震颤，以墨色气浪表现剑风

## 二、小龙女

### 2.1 古墓期

**服装描述**

| 维度 | 设计 |
|------|------|
| 款式 | 对襟长衫 + 百迭裙 + 披帛。长衫及踝，袖口微收。裙摆层叠如莲瓣。披帛绕臂垂落，是古墓派标志性装束。形制取南宋褙子之简洁，去其世俗装饰，留其清雅骨架 |
| 面料 | 冰绡（虚构面料，设定为古墓特产丝织品），质感介于绡与纱之间，半透明，有冷光微闪。触感冰凉，如寒玉凝脂 |
| 色彩 | 主色霜白（#FAFAFA），披帛淡蓝（#D6E8F0），裙摆内层冰蓝（#B8D4E3）。通体近白，在古墓冷光下泛蓝。如月色凝为人形，是"白衣如雪"的极致表达 |
| 层次 | 三层：冰蓝内衬→霜白长衫→淡蓝披帛。裙摆六幅百迭，行走时层叠开合如水波。披帛从双肩绕至臂弯垂落，静止时如两道清泉，动时如白练飞舞 |

**发型与妆容**

- 发型：长发及腰，以白玉簪束于顶成简单发髻，余发垂披。额前无碎发，发丝如缎，一丝不乱。古墓十六年，发丝从未凌乱过
- 妆容：极简至近乎无妆。肤色如雪（古墓寒玉床所致），眉如远山含黛，唇色极淡近乎透明。眼眸黑亮如深潭，瞳孔中可映出冷光。整体面容清冷，如不食人间烟火的玉人

**关键配饰/道具**

- 白玉簪：古墓派历代传承，簪头雕有寒梅纹
- 金丝手套：古墓派修炼"美女拳法"时佩戴，极薄如蝉翼，指尖处金丝织就
- 寒玉床：场景道具，古墓核心器物。整块寒玉雕成，泛幽蓝冷光

**武器设计**

- **淑女剑**：剑身银白，窄而修长，剑格处镂刻并蒂莲纹。剑穗以冰蓝丝绦制成，末端缀一颗米粒大的珍珠。轻灵如柳，与杨过的"君子剑"成对。挥动时剑光如霜雪纷飞，以淡蓝色光弧表现剑气

### 2.2 入世期（绝情谷后）

**服装描述**

| 维度 | 设计 |
|------|------|
| 款式 | 交领长衫 + 齐腰裙 + 短披风。形制从对襟改为交领，暗示从古墓出世到红尘入世的转变。短披风替代披帛，更利落实用。裙摆从百迭改为齐腰，行走更便 |
| 面料 | 从冰绡转为素绢与细纱。质感更"人间"，但仍保持清雅。纱质披风在风中飘动，保留一丝仙气 |
| 色彩 | 主色淡藕荷（#E8D5DE），外衫月白（#F0EDE5），披风烟灰蓝（#9BA8B7）。从纯白转为藕荷，是红尘留痕——不是染浊，而是沾了人间烟火的温度。色彩如水墨画中淡彩点染，克制而有余韵 |
| 层次 | 两层：月白内衫→藕荷外衫 + 烟灰蓝短披风。层次减少，是古墓清规的褪去。披风可脱卸，战斗时只着长衫 |

**发型与妆容**

- 发型：长发仍及腰，但不再束髻，改为全披散落。额前有自然碎发，是被风吹乱的痕迹。发间别一朵干花（杨过所摘），是唯一装饰
- 妆容：与古墓期相比，唇色从透明转为极淡的粉（人间饮食所致），面颊有极微的红润。眼神从深潭冷光转为含情带暖。变化极细微，是"入世而不染"的分寸

**关键配饰/道具**

- 干花发饰：杨过在山野间摘的野花，已风干，别在发间
- 情花花瓣：夹在衣襟内袋中的干枯情花花瓣，是绝情谷的记忆
- 蜂蜜瓶：小瓷瓶，装古墓玉蜂蜂蜜，是疗伤圣药

**武器设计**

- **淑女剑**（同古墓期）：剑穗更换为淡粉丝绦，与藕荷衣色呼应。剑身依旧银白，但剑光从冷蓝转为暖白，映射心境变化

## 三、郭靖

**服装描述**

| 维度 | 设计 |
|------|------|
| 款式 | 圆领长袍 + 束腰 + 披风。形制取南宋武官常服，去其官衔补子，留其端庄骨架。袍身宽大但不臃肿，披风及地。整体造型厚重稳健，如一座移动的城墙 |
| 面料 | 外袍用厚绢，披风用毛呢（北方寒地所产），内衬棉布。质感厚重扎实，与郭靖"拙"的性格呼应 |
| 色彩 | 主色赭黄（#C4A35A），披风深褐（#5C3A21），腰封玄色（#2C2C2C）。赭黄是南宋武将常用色，取"黄袍"之意而不僭越。深褐披风如大地，玄色腰封如铁壁。整体色调沉稳厚重，如水墨画中的焦墨皴擦 |
| 层次 | 三层：棉布内衫→赭黄长袍→深褐披风。腰封束于袍外，上挂刀鞘。披风内衬暗红色，翻飞时露出，是隐藏的热血 |

**发型与妆容**

- 发型：束发戴冠，网巾束发，冠为铁质朴冠（非玉非金），是郭靖不慕奢华的体现。发丝整齐，一丝不苟
- 妆容：面容方正，浓眉大眼，肤色黝黑（常年征战日晒）。眉间有竖纹（忧国忧民所致），法令纹深刻。唇色偏深，下颌宽厚。整体面相忠厚刚毅，如庙堂金刚

**关键配饰/道具**

- 铁冠：朴素铁质发冠，无宝石镶嵌
- 腰间虎符：襄阳守将兵符，铜质虎形
- 武穆遗书：贴身携带的兵法书卷，布面线装

**武器设计**

- **弓与箭**：长弓以柘木为干、牛角为附，弓身缠红绳。箭壶挂于腰间，壶中十二支狼牙箭。郭靖射雕之名，弓是第一武器
- **降龙十八掌**：无实体武器，以掌法为主。掌风以金色龙形气劲表现，龙形随掌势翻腾，是全片最壮观的武学视觉特效

## 四、黄蓉

**服装描述**

| 维度 | 设计 |
|------|------|
| 款式 | 对襟褙子 + 抹胸 + 百迭裙。形制取南宋贵妇常服，褙子及膝，裙摆曳地。剪裁合体贴身，显精明干练。袖口略收，便于书写与运筹，区别于闺阁女子的广袖 |
| 面料 | 外褙用织锦缎，裙用素罗，抹胸用细绢。织锦缎上有暗纹（凤凰穿花），需侧光才可见。面料精致但不奢靡，是"聪明人的克制" |
| 色彩 | 主色鹅黄（#F5D76E），裙碧绿（#5B8C5A），抹胸月白（#F0EDE5）。鹅黄是黄蓉标志性色彩，碧绿裙呼应"桃花岛"意象。整体色调明快而不艳俗，如春日水墨中的点彩 |
| 层次 | 三层：月白抹胸→鹅黄褙子→碧绿百迭裙。腰系丝绦，绦上挂香囊与玉佩。行走时裙摆与褙子下摆形成黄绿交叠，如柳拂春水 |

**发型与妆容**

- 发型：盘发成髻，髻式为南宋流行的"包髻"，以金簪固定。额前无碎发，一丝不乱。发间插一支金步摇，步摇坠珍珠，行动时微微晃动
- 妆容：眉形细长微挑，眼角上扬带精明。唇色朱红，面颊微施粉黛。肤色白皙但非苍白，是保养得当的中年美妇。面容秀丽中带威严，笑意中藏锋芒

**关键配饰/道具**

- 金步摇：桃花岛传承，步摇坠三颗南珠
- 打狗棒：碧玉质，杖头雕狗首，是丐帮帮主信物。黄蓉代任帮主时持有
- 香囊：鹅黄锦缎，内装桃花岛特制药粉
- 玉佩：腰间白玉佩，刻有桃花纹

## 附录：视觉风格统一规范

### 水墨底色处理

- 所有角色线稿以毛笔笔触绘制，边缘有自然枯笔效果
- 色彩以"墨分五色"为原则：焦、浓、重、淡、清
- 阴影不使用黑色，而使用对应色的深墨版本
- 高光不使用白色，而使用留白（宣纸底色透出）

### 动态布料规范

- 轻薄面料（纱、绡、绢）：飘动幅度大，轨迹如行云流水，边缘有透明渐变
- 厚重面料（毛呢、织锦、棉布）：飘动幅度小，褶皱深而少，边缘硬朗
- 披帛/飘带：独立于身体运动，有延迟跟随效果，如水中丝带

### 武学特效规范

- 剑气：以角色主色的光弧表现，弧线末端有墨色晕染
- 掌风：以角色主色的气劲表现，形态与角色性格呼应（郭靖=龙形，杨过=重浪）
- 暗器：以极细光线+尾焰表现，尾焰颜色对应毒性（蓝=剧毒，银=无毒）
- 重剑气浪：以墨色涟漪表现，涟漪扩散如水中投石
