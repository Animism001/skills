# 导演表达风格DNA研究

## 一、导演的专业术语

### 1.1 镜头语言

| 术语 | 含义 | 典型用法 |
|------|------|---------|
| 景别 | 镜头取景的范围 | 特写、近景、中景、全景 |
| 运动镜头 | 摄影机的移动 | 推、拉、摇、移、跟、升降 |
| 场面调度 | 场景内演员和摄影机的安排 | "这个镜头需要重新调度" |
| 轴线 | 摄影机和演员之间的方位关系 | "不要越轴" |
| 构图 | 画面内元素的安排 | "构图要讲究" |
| 景深 | 画面清晰的范围 | "浅景深突出主体" |
| 跳切 | 打破连续性的剪辑 | "避免跳切" |
| 匹配剪辑 | 连接相似元素的剪辑 | "用匹配剪辑转场" |

### 1.2 叙事术语

| 术语 | 含义 | 典型用法 |
|------|------|---------|
| 三幕结构 | 经典叙事框架 | "第一幕要建置" |
| 节拍 | 叙事的基本单元 | "这个节拍要有转折" |
| 弧光 | 角色的成长变化 | "角色弧光不完整" |
| 高潮 | 故事的顶点 | "这场戏是高潮" |
| 铺垫 | 为高潮做准备 | "前面要有铺垫" |
| 伏笔 | 为后续埋下的线索 | "这里要埋个伏笔" |
| 转场 | 场景之间的过渡 | "用叠化转场" |

## 二、导演的表达方式

### 2.1 视觉化表达

**用图像代替语言**：
- 分镜图
- 参考视频
- 视觉板（Mood Board）

**具体的视觉描述**：
- "镜头从他的眼睛拉出..."
- "阳光从窗户斜射进来..."
- "背景要虚化..."

### 2.2 情绪化表达

**调动情感**：
- "这场戏要让观众哭"
- "要有那种心碎的感觉"
- "让观众感受到..."

**氛围描述**：
- "压抑一点"
- "温暖一些"
- "浪漫的氛围"

### 2.3 指导性表达

**给演员的指令**：
- "再慢一点"
- "情绪再深一些"
- "想象你失去了..."

**给剧组的指令**：
- "灯光再暗一点"
- "背景换个颜色"
- "再来一条"

## 三、不同导演的风格

### 3.1 李安式表达

**温和但坚定**：
- 尊重每个人的专业
- 但坚持自己的核心要求
- 善于用问题引导

**典型表达**：
- "我有个想法，你觉得怎么样"
- "能不能试试这样..."
- "我理解你的想法，但是..."

### 3.2 张艺谋式表达

**简洁有力**：
- 直接说出要求
- 很少解释原因
- 追求视觉冲击

**典型表达**：
- "红！要更红！"
- "大！场面要大！"
- "再来一遍，更夸张"

### 3.3 王家卫式表达

**模糊但诗意**：
- 不给完整剧本
- 用感觉引导
- 捕捉即兴瞬间

**典型表达**：
- "那种感觉对不对"
- "再来一次，更迷茫一点"
- "就这样，很好"

## 四、导演的决策语言

### 4.1 肯定性语言

- "很好，就是这个"
- "非常好"
- "完美"

### 4.2 否定性语言

- "不对"
- "再来"
- "不是我要的"

### 4.3 调整性语言

- "稍微改一下"
- "这里要调整"
- "试试另一种方式"

## 五、团队沟通的表达

### 5.1 对摄影指导

**光影要求**：
- "光要更硬一些"
- "阴影要更多"
- "用侧光"

**镜头要求**：
- "用长镜头"
- "手持拍摄"
- "用广角"

### 5.2 对演员

**表演指导**：
- 示范表演
- 语言引导
- 情境描述

**情感调动**：
- 讲述背景故事
- 调动演员记忆
- 营造现场氛围

### 5.3 对制片人

**需求表达**：
- 明确需要的资源
- 解释艺术追求
- 寻求支持

**妥协表达**：
- 说明哪些可以妥协
- 坚持哪些不能变
- 寻找替代方案
