# literature-to-json-script 方法论

## 核心原则

### 1. 完整性优先
- 100%保留所有原著内容
- 不删减、不修改、不添加任何内容
- 包括所有标点符号、换行格式

### 2. 结构化标记
- 将原文内容按类型结构化标记
- 保持原文位置索引
- 便于后续Agent使用

### 3. 逐章节处理
- 每次转换一个章节/场景
- 保持章节完整性
- 便于用户逐章审核

## 识别规则

### 对话识别
- 引号内的内容识别为对话
- 识别说话人
- 保持对话原始格式

### 叙述识别
- 场景描述、环境描写
- 人物动作、心理活动
- 过渡性文字

### 角色识别
- 统计出场角色
- 记录角色出现位置
- 收集角色对话和动作
