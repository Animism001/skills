# 视频生成提示词规范

## 基本结构

```
[场景描述], [人物描述], [动作描述], [运镜方式], [情绪氛围], [光影描述], [时长]秒, [参考图片引用]
```

## 必须包含的要素

1. **场景描述** — 环境背景
2. **人物描述** — 出场人物及其状态
3. **动作描述** — 人物在做什么
4. **运镜方式** — 镜头如何运动
5. **情绪氛围** — 画面整体氛围
6. **光影描述** — 光线效果
7. **时长** — 镜头持续时间
8. **参考图片引用** — 引用上游角色的图片生成结果

## 参考图片引用格式

```
场景参考：[scene-designer/场景名/基础场景.png]
人物参考：[casting-director/角色名/基础形象.png]
服化道参考：[costume-designer/角色名/场景A服装.png]
动作参考：[action-director/动作段落/动作1.mp4]
```

## 运镜关键词

| 运镜 | 英文关键词 |
|------|-----------|
| 固定 | static shot, locked camera |
| 推 | push in, dolly in |
| 拉 | pull out, dolly out |
| 摇 | pan left/right |
| 移 | tracking shot |
| 跟 | follow shot |
| 升降 | crane shot, boom up/down |
| 手持 | handheld, shaky cam |
| 环绕 | orbit shot, 360° |

## 体裁特化

| 体裁 | 视频提示词特点 |
|------|--------------|
| 短剧 | 快切、竖屏、2-5秒/镜、强视觉冲击 |
| 微电影 | 精炼镜头、高效叙事、5-10秒/镜 |
| 电影 | 复杂运镜、长镜头、10-30秒/镜 |
| 电视剧 | 标准镜头、5-15秒/镜、场景复用 |
