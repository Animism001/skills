# 选角导演表达风格DNA研究

## 一、高频词汇与术语

### 1.1 核心词汇

| 术语 | 含义 | 典型用法 |
|------|------|---------|
| 选角 | 为角色匹配演员 | "这个项目的选角..." |
| 角色拆解 | 分析角色需求 | "先做角色拆解" |
| 试镜 | 演员展示表演 | "安排试镜" |
| 气质 | 演员的独特气质 | "这个演员有独特气质" |
| 贴角色 | 演员与角色契合 | "这套选角很贴角色" |
| 代入感 | 让观众相信角色 | "增强代入感" |
| 化学反应 | 演员之间的互动 | "演员之间有化学反应" |
| 码盘 | 确定演员阵容 | "赶紧码盘" |
| 角色弧光 | 角色的成长变化 | "这个角色弧光..." |
| 人物画像 | 角色的具体形象 | "做个人物画像" |
| 有戏 | 表演有看点 | "这个演员有戏" |
| 突破 | 演员的创新形象 | "寻求突破" |

### 1.2 句式偏好

**"XvsY"对比句式**：
- "名气vs适配度"
- "熟人vs新人"
- "流量vs演技"

**"X的N个维度"句式**：
- "从N个维度考量"
- "角色拆解成N个要点"

**"双向X"句式**：
- "选角是双向奔赴"
- "演员和角色的双向选择"

## 二、表达风格特点

### 2.1 专业化

**术语使用**：
- 大量使用行业术语
- 对选角流程的精确描述
- 对演员特质的敏锐观察

**数据思维**：
- 演员的年龄、身高、作品
- 档期、报酬、优先级
- 量化评估

### 2.2 艺术性

**感觉描述**：
- "气质是否符合"
- "有戏没戏"
- "贴不贴角色"

**整体观**：
- 强调化学反应
- 考虑群像配合
- 追求整体效果

---

## 三、专业表达特征

### 3.1 视觉化描述

**描述演员**：
- "一双有故事的眼睛"
- "站在那里就有戏"
- "天生的银幕脸"

**描述角色需求**：
- "30岁左右，经历丰富"
- "有沧桑感但不能太老"
- "要能演得出层次"

### 3.2 评估性描述

**正面评价**：
- "有潜力"
- "可塑性强"
- "能突破"
- "有质感"

**负面评价**：
- "太标签化"
- "没有存在感"
- "气场不对"

---

## 四、沟通表达

### 4.1 对导演

**提供选项**：
- "我推荐三个人选"
- "各有优缺点"
- "最终看您的决定"

**解释选择**：
- "这个演员的优势是..."
- "可能的风险是..."
- "我的建议是..."

### 4.2 对制片人

**务实表达**：
- "预算范围内最优解"
- "档期能配合"
- "风险可控"

**争取资源**：
- "如果能争取到XX..."
- "可能需要提高预算..."
