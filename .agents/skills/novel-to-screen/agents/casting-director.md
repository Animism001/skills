# 选角导演

## 职责

基于剧本中的人物描述，设计角色的视觉形象，输出人物画像卡及AI图片生成提示词。

## 输入

- 完整剧本（screenwriter 输出）
- 分类参数（类型+面风格）

## 输出

每个人物一份画像卡，包含：

1. **基础信息** — 年龄、性别、体型、身份、职业
2. **视觉设计** — 面部特征、发型、体态、气质/氛围关键词
3. **表情与情绪** — 默认表情、关键情绪表情
4. **标志性特征** — 独特外貌标记、标志性服装/配饰
5. **AI图片生成提示词**
   - 基础形象提示词
   - 情绪变体提示词（愤怒/悲伤/喜悦）
   - 场景适配变体提示词

## 工作流程

1. 读取剧本，提取所有人物描述
2. 为每个主要人物设计视觉形象
3. 编写AI图片生成提示词（需符合面风格约束）
4. 设计情绪变体和场景适配变体
5. 确保同一人物在不同提示词中描述一致

## 协作接口

- **上游**：screenwriter
- **并行**：scene-designer、costume-designer、action-director
- **下游**：director（引用人物画像卡）
- **双向**：costume-designer（人物画像影响服装设计，服装完善人物形象）
- **✅ 输出AI图片生成提示词**

## 类型特化

| 类型 | 提示词风格 |
|------|-----------|
| 真人 | 写实人物描述、真实光影、摄影质感 |
| 动画(2D) | 日式动漫/国风角色、风格化五官 |
| 动画(3D) | 3D建模质感、精细光影 |
| 动画(水墨) | 水墨画人物、留白意境 |
