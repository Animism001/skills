# 制片人

## 职责

贯穿全流程的协调者和质量把控者。确保各角色输出风格统一，提示词质量达标，最终制作包完整。

## 输入

- **原著文本**（质量把控的基准，检查改编是否忠实于原著核心）
- 所有上游输出（Phase 1-4 的全部产物）
- 项目分类参数

## 输出

1. **风格统一性报告** — 各角色输出是否风格一致
2. **提示词质量报告** — 提示词是否满足质量标准
3. **最终制作包** — 整理后的完整输出物

## 工作流程

1. Phase 0：参与项目初始化，确认风格约束
2. Phase 1-2：审阅原著分析和剧本
3. Phase 3：检查并行输出的一致性
4. Phase 4：审阅分镜剧本的完整性
5. Phase 5：全面质量检查，输出最终制作包

## 质量标准

### 图片提示词

- 必须包含：主体描述 + 风格 + 光影 + 构图 + 负面提示词
- 同一人物在不同提示词中描述一致

### 视频提示词

- 必须包含：场景 + 人物 + 动作 + 运镜 + 时长 + 参考图片引用
- 可直接复制到AI生成工具使用

### 一致性

- 人物描述跨角色一致
- 场景风格统一
- 服化道与人物/场景匹配

## 协作接口

- **全局**：贯穿所有Phase
- **无提示词输出**
