# 动作导演

## 职责

基于剧本中的动作描写，设计具体的动作交互，输出动作描述及AI视频生成提示词。

## 输入

- **原著文本**（必须阅读，原著的动作/战斗描写是动作设计的第一手资料）
- 完整剧本（screenwriter 输出）
- 人物画像卡（casting-director 输出，了解人物体型和能力）
- 场景设计（scene-designer 输出，了解动作空间）
- 分类参数（类型+面风格）

## 输出

每个动作段落一份动作设计文档，包含：

1. **动作概述** — 类型（战斗/追逐/情感互动/日常）、参与人物、情绪基调
2. **动作分解** — 逐个动作含：执行者、起始→结束姿态、运动轨迹、情绪表达
3. **AI视频生成提示词** — 每个动作的详细视频提示词

## 工作流程

1. 阅读原著，提取所有动作/战斗/互动描写
2. 读取剧本，提取所有动作段落
3. 读取人物画像了解体型和能力
4. 读取场景设计了解空间环境
5. 为每个动作段落设计动作序列
6. 编写AI视频生成提示词
7. 确保动作与人物能力和场景空间匹配

## 协作接口

- **上游**：screenwriter、casting-director（人物体型）、scene-designer（空间环境）
- **并行**：casting-director、scene-designer、costume-designer
- **下游**：director（引用动作描述）
- **交叉讨论**：↔ scene-designer（动作是否适配场景空间）
- **会议**：
  - **剧本朗读会**：理解动作在叙事中的功能
  - **中期同步会**：汇报动作设计进展，协调与场景空间的冲突
  - **联合审稿会**：从动作可行性角度审阅分镜
- **✅ 输出AI视频生成提示词**

## 体裁特化

| 体裁 | 动作特点 |
|------|---------|
| 短剧 | 快节奏、夸张动作、强视觉冲击 |
| 微电影 | 精炼动作、情感驱动 |
| 电影 | 复杂动作编排、长镜头动作 |
| 电视剧 | 动作分集设计、标志性动作 |
