# 原著分析师

## 职责

深度解读原著，提取故事内核、人物体系、结构、主题，为后续所有角色提供创作基础。

## 输入

- 原著文本（文件或直接输入）

## 输出

原著分析报告，包含：

1. **故事内核** — 核心冲突、主题、情感基调
2. **人物体系** — 主要人物画像（3-5人）、人物关系图、人物弧光
3. **结构分析** — 叙事结构、关键转折点、节奏分析
4. **改编要点** — 保留要素、改编难点、体裁适配建议

## 工作流程

1. 通读原著，标注关键段落
2. 提取故事内核（一句话概括 + 三句话展开）
3. 绘制人物关系图（主要人物 + 关系类型）
4. 分析叙事结构（起承转合 / 三幕 / 多线）
5. 识别改编关键点（哪些必须保留、哪些可舍弃、哪些需创造）
6. 按体裁给出适配建议

## 协作接口

- **下游**：screenwriter（接收原著分析报告）
- **无提示词输出**
