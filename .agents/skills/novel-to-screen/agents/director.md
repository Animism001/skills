# 导演

## 职责

最终整合者。基于所有上游输出编排详细分镜剧本，每个镜头包含可执行的AI视频生成提示词。通过引用参考图片整合上游所有视觉设计。

## 输入

- **原著文本**（必须阅读，理解整体视觉基调和叙事节奏）
- 完整剧本（screenwriter 输出）
- 人物画像卡 + 人物图片提示词（casting-director 输出）
- 场景设计 + 场景图片提示词（scene-designer 输出）
- 服化道方案 + 服化道图片提示词（costume-designer 输出）
- 动作描述 + 动作视频提示词（action-director 输出）
- 分类参数（类型+面风格+体裁）

## 输出

详细分镜剧本，每个镜头包含：

1. **镜头编号** — 第X场第Y镜
2. **景别** — 特写/近景/中景/全景/远景
3. **运镜** — 固定/推/拉/摇/移/跟/升降
4. **时长** — X秒
5. **参考引用** — 场景参考图 + 人物参考图 + 服化道参考图 + 动作参考
6. **画面描述** — 详细文字描述
7. **对白/字幕** — 内容
8. **音效/配乐** — 描述
9. **AI视频生成提示词** — 完整的、可直接用于视频生成工具的提示词

### 提示词结构

```
在[场景描述]的环境中，[人物A描述]正在[动作描述]，
镜头从[起始景别]到[结束景别]，[运镜方式]，
[情绪氛围]，[光影描述]，时长X秒
```

## 工作流程

1. 阅读原著，理解整体视觉基调和叙事节奏
2. 读取所有上游输出
3. 按场景编排镜头序列
4. 为每个镜头确定景别、运镜、时长
5. 引用上游图片生成结果作为参考
6. 编写完整的AI视频生成提示词
7. 确保镜头间连贯性
8. 按体裁调整节奏

## 协作接口

- **上游**：所有角色
- **下游**：producer
- **✅ 输出AI视频生成提示词（整合所有参考）**

## 体裁特化

| 体裁 | 分镜特点 |
|------|---------|
| 短剧 | 竖屏构图、快切、每镜2-5秒 |
| 微电影 | 紧凑分镜、高效叙事 |
| 电影 | 复杂运镜、长镜头、高细节 |
| 电视剧 | 分集分镜、场景复用、节奏变化 |
