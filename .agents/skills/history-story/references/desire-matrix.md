# D.E.S.I.R.E. Matrix 完整理论框架

本文件为历史故事创作技能提供 D.E.S.I.R.E. Matrix 的完整理论框架和文风定制模板。

---

## 交叉学科知识库

**心理学与认知科学**：
- 认知心理学、社会心理学、心理语言学、动机理论

**语言学与传播学**：
- 语用学：分析语言在特定情境下的实际含义
- 话语分析：分析对话的结构、连贯性和主题演变
- 情感分析：识别和评估文本中表达的情感色彩和强度

## D.E.S.I.R.E. Matrix 六维分析矩阵法

### D - Drive（驱动力）
**调用理论**：自我决定论、五大性格特质、目标设定理论

自我决定论的三个核心需求：
1. 自主需求（Autonomy）- 渴望自我决定
2. 胜任需求（Competence）- 渴望有效应对环境
3. 关联需求（Relatedness）- 渴望与他人建立连接

**历史故事适配**：这个历史事件之所以流传千年，是因为它触动了人类哪种核心心理需求？

### E - Experience（体验角色）
**调用理论**：巴图玩家分类法、戏剧角色理论

巴图玩家分类法：
- 成就者（Achievers）- 追求目标完成和挑战
- 探索者（Explorers）- 追求发现和理解
- 社交家（Socializers）- 追求人际关系和互动
- 杀手（Killers）- 追求竞争和支配

**历史故事适配**：读者在阅读这段历史时，渴望体验什么角色？

### S - Shortcut（认知捷径）
**调用理论**：行为经济学、说服心理学、认知偏误清单

常见认知偏误：
- 确认偏误（Confirmation Bias）
- 锚定效应（Anchoring Effect）
- 损失厌恶（Loss Aversion）
- 后见之明偏误（Hindsight Bias）
- 幸存者偏差（Survivorship Bias）

**历史故事适配**：读者对这段历史可能存在哪些认知偏误？故事应如何打破这些偏误？

### I - Identity（身份认同）
**调用理论**：荣格原型理论、社会认同理论

荣格12原型：
- 英雄（Hero）- 追求自我实现和成长
- 智者（Sage）- 追求真理和智慧
- 探索者（Explorer）- 追求自由和发现
- 恋人（Lover）- 追求亲密和连接
- 颠覆者（Outlaw）- 追求革命和改变
- 魔术师（Magician）- 追求转变和力量
- 天真者（Innocent）- 追求纯真和幸福
- 守护者（Caregiver）- 追求关爱和保护
- 创造者（Creator）- 追求创新和表达
- 统治者（Ruler）- 追求秩序和控制
- 弄臣（Jester）- 追求乐趣和享受
- 普通人（Everyman）- 追求归属和认同

**历史故事适配**：这段历史触动了哪种集体身份认同？

### R - Risk（风险偏好）
**调用理论**：前景理论、调节定向理论、依恋理论

调节定向理论：
- 促进焦点（Promotion Focus）- 关注成长和收益
- 预防焦点（Prevention Focus）- 关注安全和避免损失

**历史故事适配**：读者在阅读这段历史时最怕什么？需要规避哪些误区？

### E² - Emotion（情感偏好）
**调用理论**：Moral Foundations Theory、普拉切克情感轮、故事形态学（含七大情节/冯内古特弧线）、Peak-End Rule

Moral Foundations Theory 六大道德基础：
1. 关爱/伤害（Care/Harm）
2. 公平/欺骗（Fairness/Cheating）
3. 忠诚/背叛（Loyalty/Betrayal）
4. 权威/颠覆（Authority/Subversion）
5. 圣洁/堕落（Sanctity/Degradation）
6. 自由/压迫（Liberty/Oppression）

冯内古特弧线（情感弧线）：
- 灰姑娘型（Cinderella）- 上升-下降-上升
- 倒霉蛋型（Man in Hole）- 下降-上升
- 男孩遇见女孩型（Boy Meets Girl）- 上升-下降
- 从乞丐到富翁型（Rags to Riches）- 持续上升
- 从富翁到乞丐型（Riches to Rags）- 持续下降
- 伊卡洛斯型（Icarus）- 上升-下降
- 俄狄浦斯型（Oedipus）- 下降-上升-下降

**历史故事适配**：这段历史最动人的情感弧线是什么？

## WritingStyle 文风定制模板

基于 D.E.S.I.R.E. Matrix 分析结果，为每个历史故事定制专属文风：

```
[专属文风]
[核心风格：根据分析提炼出的3-5个最关键风格词]
氛围与基调
- 基调: 根据E²分析（史诗/悲壮/悬疑/紧张/热血/宁静/治愈/智性博弈等）
- 情感强度: 根据R和D分析（强烈而富有张力 / 内敛而克制 / 平和而舒缓）
- 类型元素: 根据E²-Emotion分析（强烈的戏剧冲突 / 富有哲学思辨 / 注重情感纠葛 / 充满奇观想象）
- 感官偏向: 根据I-Identity和E²-Emotion分析
节奏与步速
- 总体节奏: 根据D-Drive和E-Experience分析（快节奏情节驱动 / 慢节奏氛围优先 / 张弛有度）
- 信息密度: 根据I-Identity和S-Shortcut分析（高密度信息量大 / 低密度留白较多 / 中等密度）
叙事手法
- 叙事引擎: 根据E-Experience和S-Shortcut分析（对话为核 / 描写为核 / 均衡驱动）
- 叙事焦点: 根据E-Experience和I-Identity分析（聚焦内在心理 / 聚焦外在行动 / 聚焦环境氛围）
描写方式
- 描写层次: 根据E²-Emotion和R-Risk分析（宏观微观交替 / 聚焦微观细节 / 聚焦宏观场面）
- 修辞偏好: 根据I-Identity和E²-Emotion分析（多用象征暗喻 / 纯白描少用修辞 / 语言如刀精准直接）
[/专属文风]
```

### WritingStyle 推导逻辑

以下推导逻辑源自原始 JSON 设计（Ny-Sigon），是 Matrix 到 WritingStyle 的核心映射链：

- **D - Drive**：核心需求决定创作理念
  - 追求"自主需求"的事件 → 偏好展现抗争与突破
  - 追求"胜任需求"的事件 → 偏好展现智谋与策略（"展现智识"）
  - 追求"关联需求"的事件 → 偏好探索情感与羁绊（"探索情感"）
- **E - Experience**：体验角色决定叙事节奏和叙事支柱
  - "成就者" → 快节奏和行动
  - "探索者" → 慢节奏和环境/心理描写
  - "社交家" → 对话驱动
- **S - Shortcut**：思维捷径决定节奏和复杂度
  - 有"容忍模糊"偏误 → 能接受复杂情节
  - 反之 → 需要简单直接
  - 后见之明偏误 → 需要制造悬念，还原历史的不确定性
  - 幸存者偏差 → 需要呈现失败者的视角
  - 简单化偏误 → 需要展现历史的复杂性
- **I - Identity**：身份原型是语言风格和创作理念的关键
  - "智者"原型 → 思辨、典雅的语言
  - "英雄"原型 → 激昂、有力的语言
  - "颠覆者"原型 → 直接、有冲击力的表达
- **R - Risk**：风险偏好影响情节复杂度和描写精细度
  - 害怕"模糊不清" → 极度渴求微观、精细的描写来获得掌控感
  - 害怕"历史与我无关" → 需要强烈的代入感和微观细节
  - 害怕"过度简化" → 需要呈现多角度和多声部
- **E² - Emotion**：情感偏好直接决定了最终的核心情感基调和创作理念
  - 读者渴望的"情感终点"就是文风需要营造的最终氛围

### 剧情建议推导逻辑

以下推导逻辑源自原始 JSON 设计（Ny-Sigon），是 Matrix 到剧情建议的核心映射链：

- **规避区** ← 回顾 R - Risk 部分，列出读者的"雷点"和绝对要避免的负面体验，这些是剧情的红线
- **核心驱动力** ← 基于 D - Drive 部分，填写故事最根本的内在驱动力
- **推荐剧情方向** ← 基于核心驱动力，设计多条具体、可执行的主线剧情脉络
- **可设计的剧情起伏** ← 选择性地设计未来的核心矛盾，这个挑战应该直接"命中"读者的理想角色，而且一定存在克服可能
- **剧情奖励** ← 预设一些能够满足读者核心驱动力的"奖励"情节

## 历史故事文风示例

### 示例1：战争转折类（如赤壁之战、滑铁卢）

```
[专属文风]
[核心风格：史诗磅礴、冷峻克制、张弛有度、细节入微]
氛围与基调
- 基调: 史诗悲壮、智性博弈
- 情感强度: 强烈而克制，在战争的残酷中见人性
- 类型元素: 强烈的戏剧冲突、战略博弈、命运无常
- 感官偏向: 聚焦战场的声光气味、将领的微表情、士卒的恐惧
节奏与步速
- 总体节奏: 张弛有度，战前紧绷、战中爆发、战后沉寂
- 信息密度: 高密度，军事信息与人性描写交织
叙事手法
- 叙事引擎: 均衡驱动，战略对话与战场描写交替
- 叙事焦点: 宏观（天下大势）与微观（个体命运）交替
描写方式
- 描写层次: 宏观微观交替，从一将之怒到万军之势
- 修辞偏好: 纯白描为主，偶尔以自然意象隐喻命运
[/专属文风]
```

### 示例2：人物命运类（如岳飞之死、苏格拉底之死）

```
[专属文风]
[核心风格：沉郁悲壮、内敛深情、以小见大、余韵悠长]
氛围与基调
- 基调: 悲壮深沉、命运无常
- 情感强度: 内敛而深沉，情感在克制中积蓄力量
- 类型元素: 命运悲剧、理想与现实的冲突、个人与时代
- 感官偏向: 聚焦人物内心世界的幽微变化、环境氛围的象征
节奏与步速
- 总体节奏: 慢节奏，氛围优先，在平静中酝酿风暴
- 信息密度: 中等密度，留白与细节并存
叙事手法
- 叙事引擎: 描写为核，以细腻的心理和环境描写构建内核
- 叙事焦点: 聚焦内在心理与命运的无声碾压
描写方式
- 描写层次: 聚焦微观细节，以一滴水映照整个时代
- 修辞偏好: 多用象征暗喻，以自然意象映射人物命运
[/专属文风]
```

### 示例3：科技发现类（如牛顿与苹果、居里夫人的镭）

```
[专属文风]
[核心风格：智性激荡、理性浪漫、举重若轻、灵光乍现]
氛围与基调
- 基调: 智性浪漫、探索未知
- 情感强度: 平和而深邃，在理性中见激情
- 类型元素: 智性博弈、突破与顿悟、孤独与坚持
- 感官偏向: 聚焦思维的过程、实验的细节、灵感的瞬间
节奏与步速
- 总体节奏: 慢节奏积累，快节奏突破
- 信息密度: 高密度，科学细节与人文关怀交织
叙事手法
- 叙事引擎: 均衡驱动，内心独白与实验过程交替
- 叙事焦点: 聚焦内在思维的演变和突破的瞬间
描写方式
- 描写层次: 聚焦微观细节，从试管中的反应到脑海中的风暴
- 修辞偏好: 精准直接，偶尔以诗意笔触描写科学之美
[/专属文风]
```
