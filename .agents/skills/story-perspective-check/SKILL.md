# 换位思考验证

创作者最危险的盲区是"自己懂了就以为读者也懂"。你脑中有完整的故事世界，读者只有你写出来的文字。换位思考验证就是让你脱下创作者的帽子，戴上读者的眼镜，从零开始审视自己的作品。

## 核心原则

**信息不对称**：你知道一切，读者只知道你写的。每次验证都要假设读者对你脑中的设定一无所知。

## 三层验证

### 第一层：可理解性验证

**问题**：读者能否跟上故事的逻辑？

检验方法：
1. **遮住脑补测试**：只看文字本身，不借助你脑中的补充信息，情节是否连贯？
2. **陌生人测试**：把故事给一个完全不了解背景的人看，TA能否理解发生了什么？
3. **时间线测试**：读者能否理清事件的发生顺序？（尤其是非线性叙事）

常见问题：
- 关键信息只在作者脑中，没有写出来
- 人物动机不明确——读者不理解为什么角色要这么做
- 时间跳跃没有提示，读者迷失了

**输出**：
```
### 可理解性检查
| 情节点 | 读者能否理解？ | 缺失信息 | 修复方案 |
|--------|----------------|----------|----------|
| ... | ✅/❌ | | |
```

### 第二层：情感共鸣验证

**问题**：读者是否在乎故事中发生了什么？

检验方法：
1. **赌注测试**：如果主角失败，读者是否感到遗憾？如果不遗憾，赌注不够高
2. **共鸣测试**：读者能否在主角身上看到自己的影子？主角的困境是否具有普遍性？
3. **意外测试**：故事的转折是否既出乎意料又在情理之中？如果读者早已猜到，悬念不足

三个关键情感时刻的检验：

| 时刻 | 应有的情感 | 检验问题 |
|------|-----------|----------|
| 触发事件 | 震动/好奇 | 读者是否想继续看下去？ |
| 危机时刻 | 紧张/焦虑 | 读者是否为主角捏一把汗？ |
| 高潮 | 释放/感动 | 读者是否感到情感被满足？ |

**输出**：
```
### 情感共鸣检查
| 情感时刻 | 目标情感 | 实际效果 | 差距原因 | 修复方案 |
|----------|----------|----------|----------|----------|
| 触发事件 | | | | |
| 危机时刻 | | | | |
| 高潮 | | | | |
```

### 第三层：注意力验证

**问题**：读者是否始终有理由继续读下去？

检验方法：
1. **钩子测试**：每一段落的结尾是否让读者想看下一段？
2. **承诺测试**：故事开头是否做出了某种承诺？结尾是否兑现了？
3. **冗余测试**：是否有段落删掉后不影响理解和情感？如果有，这些段落拖慢了节奏

注意力曲线：
```
读者注意力
  ↑
  │    ★    ★
  │   ╱ ╲  ╱ ╲    ★
  │  ╱   ╲╱   ╲  ╱ ╲
  │ ╱          ╲╱   ╲
  │╱                  ╲
  └─────────────────────→ 阅读进度
```

**注意力杀手**：
- 信息倾倒（一次性给太多背景信息）
- 无关支线（与主线无关的情节）
- 重复表达（同一个意思说了多遍）
- 缺乏悬念（读者没有"然后呢？"的冲动）

**输出**：
```
### 注意力检查
| 段落 | 钩子类型 | 是否有理由继续 | 问题 | 修复 |
|------|----------|----------------|------|------|
| 开头 | | | | |
| 中段1 | | | | |
| 中段2 | | | | |
| 高潮 | | | | |
| 结尾 | | | | |
```

## 完整验证报告模板

```
## 换位思考验证报告

### 模拟读者画像
- 类型：[目标读者类型]
- 期待：[读者对这类故事的期待]
- 耐心：[读者愿意给多少时间进入故事]

### 第一层：可理解性
- 总评：✅/⚠️/❌
- 关键问题：...
- 修复优先级：...

### 第二层：情感共鸣
- 总评：✅/⚠️/❌
- 最打动人的时刻：...
- 最缺乏共鸣的时刻：...
- 修复优先级：...

### 第三层：注意力
- 总评：✅/⚠️/❌
- 最可能弃读的位置：...
- 最吸引人的段落：...
- 修复优先级：...

### 综合诊断
- 最需要修复的1个问题：...
- 最值得保留的1个亮点：...
- 修复后的预期效果：...
```

## 与其他技能的配合

- 验证前用 **story-seven-voices**（七声部框架）做全面诊断
- 注意力问题用 **story-rhythm**（节奏对比组合）调整节奏
- 可理解性问题用 **story-three-pillars**（三支柱八要素）检查基础
- 情感共鸣问题用 **story-character-design**（人物设计五步法）深化人物
