# 全局终检（增强版）

完成所有 Phase 后的最终检查。核心原则：**逻辑不能断，动力不能停，换位思考验证效果**

## 一、逻辑与动力检查

- [ ] **逻辑性**：从头到尾因果链无断裂
- [ ] **动力性**：故事始终有推进力，无停滞段落
- [ ] **反差**：开篇与高潮有足够的状态距离
- [ ] **人物弧光**：主角在身份/欲望/动作上有可感知的变化
- [ ] **主题表达**：正负价值的对抗有明确结果
- [ ] **节奏**：有起有落，约15分钟一个路标
- [ ] **关注线**：全程有东西抓人，无长时间走神区
- [ ] **人物可信**：主要人物通过合理性+合情性检验

## 二、换位思考三层验证

### 第一层：可理解性
- [ ] **遮住脑补测试**：只看文字本身，情节是否连贯？
- [ ] **陌生人测试**：完全不了解背景的人能否理解发生了什么？
- [ ] **时间线测试**：读者能否理清事件的发生顺序？

### 第二层：情感共鸣
- [ ] **触发事件**：读者是否想继续看下去？
- [ ] **危机时刻**：读者是否为主角捏一把汗？
- [ ] **高潮**：读者是否感到情感被满足？

### 第三层：注意力
- [ ] **钩子测试**：每一段落的结尾是否让读者想看下一段？
- [ ] **承诺测试**：故事开头是否做出了某种承诺？结尾是否兑现了？
- [ ] **冗余测试**：是否有段落删掉后不影响理解和情感？

## 五、人物关系最终检查

- [ ] 每个主要人物至少有3种关系类型（处事/利益/情感/价值）
- [ ] 至少有一对关系存在不对称性（戏剧张力来源）
- [ ] 核心冲突必须是价值关系（价值不能谈判）
- [ ] 关系有弧线（没有变化的关系是死关系）
