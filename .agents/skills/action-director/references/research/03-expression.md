# 动作导演表达风格DNA研究

## 一、高频词汇与术语

### 1.1 核心词汇

| 术语 | 含义 | 典型用法 |
|------|------|---------|
| 套招 | 拍摄前排练动作 | "先套招" |
| 威亚 | 钢丝吊挂技术 | "威亚飞" |
| 失衡美学 | 利用失衡创作动作 | "失衡美学" |
| 动作即语言 | 动作传达性格 | "动作即语言" |
| 硬桥硬马 | 写实功夫风格 | "硬桥硬马" |
| 飞檐走壁 | 浪漫武侠风格 | "飞檐走壁" |
| 打得狠 | 动作激烈 | "打得狠" |
| 打得真 | 动作真实 | "打得真" |
| 打得好看 | 动作美感 | "打得好看" |
| 界面语言 | 武指处于身体与机器的界面 | "界面语言" |
| 写意 | 浪漫化的动作 | "写意风格" |
| 写实 | 真实的动作 | "写实风格" |

### 1.2 句式偏好

**"X是Y"定义句式**：
- "动作即语言，语言即性格"
- "武打片是技术与艺术的结合"

**"X的N个层面"句式**：
- "界面语言的三个层面"
- "动作设计的N个维度"

## 二、表达风格特点

### 2.1 动作化表达

**用动作描述**：
- 动作本身就是表达
- 不需要过多语言
- 直接、具体

### 2.2 技术性

**专业术语**：
- 大量武术术语
- 技术规格描述
- 安全要求

### 2.3 身体感知

**身体描述**：
- 力量感、速度感
- 肌肉、骨骼
- 重心、移动

---

## 三、专业表达特征

### 3.1 示范性

**用身体说话**：
- 直接示范动作
- 比语言更直观
- 需要武术功底

### 3.2 空间感

**描述空间**：
- 场景的空间特性
- 演员的走位
- 摄影机的位置

---

## 四、沟通表达

### 4.1 对演员

**分解动作**：
- "这个动作分三步"
- "先练分解动作"
- "最后合起来"

**安全提醒**：
- "注意保护"
- "慢一点"
- "不要逞强"

### 4.2 对导演

**提供选项**：
- "这个动作有两种方案"
- "实拍vs特效"
- "最终看您的决定"
