# Phase 13: 场景织体

> 目标：以场景为单位选择与安排内容

## 核心方法：节奏=对比组合

### 核心公式

```
节奏 = 对比 + 组合

对比 = 两种相反状态的并置（快/慢、张/弛、悲/喜、大/小）
组合 = 对比的有规律排列
```

### 六种基本对比

| 对比类型 | 快 | 慢 |
|----------|-----|-----|
| 快与慢 | 短句、密集事件 | 长句、内心独白 |
| 张与弛 | 悬念、危机 | 安全、幽默 |
| 悲与喜 | 失去、失败 | 获得、成功 |
| 大与小 | 宏大场景 | 私密时刻 |
| 聚与散 | 多线汇聚 | 分头行动 |
| 虚与实 | 梦境、回忆 | 现实、真相 |

### 五种组合模式

| 模式 | 结构 | 适用 |
|------|------|------|
| 呼吸式 | 慢→快→慢→快 | 最通用的基础节奏 |
| 递进式 | 弛→张→弛→X张 | 后半段加速的故事 |
| 钟摆式 | 悲→喜→悲→喜→悲悲 | 情感浓度高的故事 |
| 漏斗式 | 散→散→聚→聚聚聚 | 多线叙事最终汇聚 |
| 回旋式 | A→B→A'→B'→A'' | 主题反复探讨 |

### 场景织体三重奏

1. **场景线**：调整场景顺序影响信息传递、情绪表达和节奏
2. **目标线**：检查场景目标是否在故事正负价值之间运行
3. **关注线**：审视关注点连续性，安排速度/强度/长度/密度对比组合

### 场景粘合手段

- **导入**：为路标事件铺垫前置场景
- **强化**：用多个场景叠加强化同一效果
- **并列**：用平行场景展示多线
- **烘托**：用次要场景衬托主要场景
- **对比**：用反差场景增强冲击

## 操作流程

1. 纵向编织：依据序列功能+路标事件+表达目标选择场景
2. 横向编织三重奏：场景线+目标线+关注线
3. 设计关注点：奇特、有趣、刺激、移情

## 检验问题

- 节奏是否有起有落？
- 对比是否有规律（不是随机起伏）？
- 高潮是否是情绪的最高点？
- 是否有"暴风前的宁静"？
