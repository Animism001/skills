# Phase 9: 高潮大纲

> 目标：率先锁定高潮，为全篇锚定目标

## 核心方法：逆向设计法

**核心原理**：先定高潮，再反推全局。

**设计逻辑**：
```
高潮（终点）→ 危机时刻 → 核心冲突 → 触发事件 → 日常（被打破的平衡）
```

### 操作步骤

**Step 1：设计高潮**

高潮必须同时满足三个条件：
1. **主题揭示**：主角的选择直接回答了核心命题
2. **情感峰值**：读者情绪的最高点
3. **不可逆转**：高潮之后，主角的世界永远不同了

**Step 2：设计危机时刻**

危机时刻是高潮前的至暗时刻——主角面临两难选择。
- 两个选项都必须有巨大的代价
- 无论选哪个，都要失去重要的东西

**Step 3：反推核心冲突**

从高潮和危机反推：是什么力量把主角推到了这个境地？
- 核心冲突通常来自对手（人物）或环境（困境）
- 冲突的升级路径：小摩擦 → 正面交锋 → 全面对抗 → 危机 → 高潮

**Step 4：反推触发事件**

什么事件打破了日常？
- 触发事件必须不可逆
- 必须与高潮有因果链

**Step 5：反推日常**

高潮揭示了主题，日常必须呈现主题的反面。

## 操作流程

1. 用"人物/原因/动作/内容/反应"五列表格记录高潮进程
2. 识别每个环节的主动人物，划分"动作回合"
3. **逆向设计法**：先写高潮再反推

## 常见问题修复

| 问题 | 诊断 | 修复 |
|------|------|------|
| 高潮平淡 | 高潮没有主题揭示 | 重新问：主角的选择回答了什么问题？ |
| 危机不成立 | 两个选项不是真正的两难 | 加大至少一个选项的代价 |
| 冲突升级断裂 | 某一步缺乏因果 | 补充过渡事件 |
| 触发事件可逆 | 主角可以回到日常 | 增加不可逆的后果 |
| 日常与高潮无反差 | 主角没有弧线 | 重新设计日常 |

## 检验问题

- 高潮是否同时满足三个条件（主题揭示/情感峰值/不可逆）？
- 危机时刻是否真正的两难选择？
- 从高潮反推到日常的因果链是否通顺？
