# Phase 5: 故事梗概

> 目标：300-500字展现故事主要内容与结构方式

## 核心方法：递进写作

### 递进步骤

1. **一句话故事** → 核心概念（15-30字）
2. **简洁表述** → 扩展到50-100字
3. **完整梗概** → 300-500字

### 梗概内容

完整梗概应包含：
- 主角及其初始状态
- 触发事件
- 主要冲突的发展
- 高潮
- 结局与主题揭示

## 操作流程

1. 从一句话故事开始
2. 扩展为简洁表述
3. 连接分幕方案表内容为完整梗概
4. 写完后"退后一步"宏观审视：构图、布局、比例

## 产出

故事梗概（300-500字）

## 检验问题

- 是否包含所有关键情节点？
- 主角是否有清晰的变化弧线？
- 主题是否通过故事而非说教呈现？
- 长度是否适中（不太简略也不太冗长）？
