# Phase 12: 故事大纲

> 目标：写出既看"讲什么"又看"怎么讲"的大纲

## 核心方法

### 大纲原则

1. **以主角为中心**：取舍事件与人物介绍都围绕主角展开
2. **通过取舍与侧重调控节奏**：决定详略
3. **细节选"很重要"或"有意思"的**：不重要的细节省略
4. **保持画面感与简洁**：用具体场景代替抽象描述

### 大纲内容

- 主要情节线（按时间顺序）
- 关键情节点
- 人物弧线
- 主题揭示方式

## 操作流程

1. 以主角为中心取舍事件与人物介绍
2. 通过取舍与侧重调控节奏
3. 细节选"很重要"或"有意思"的，抒情仅适度
4. 保持画面感与简洁
5. 可附人物小传和背景介绍

## 产出

故事大纲

## 检验问题

- 是否以主角为中心？
- 节奏是否控制得当（详略有致）？
- 是否保持画面感？
- 是否清晰展现人物弧线和主题？
