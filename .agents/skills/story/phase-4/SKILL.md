# Phase 4: 关键事件与幕布局

> 目标：搭建故事骨架

## 核心方法：三幕四部分布局

### 布局结构

采用**三幕四部分**布局：
- 建置 → 进展 → 转折 → 解决

### 三个关键事件

用三个**关键事件**划分幕：

| 事件 | 位置 | 功能 |
|------|------|------|
| **激励事件** | 第一幕结尾 | 引出故事主体，不可逆转 |
| **转折事件** | 第二幕中点 | 障碍力变或向变，重新燃起冲突 |
| **危机事件** | 第二幕结尾 | 将主角逼入绝境 |

### 关键事件标准

每个关键事件需同时满足：
- **逻辑性**：顺理成章、不可逆
- **动力性**：决定性推动力

## 操作流程

1. 采用三幕四部分布局
2. 用三个关键事件划分幕
3. 每个关键事件满足逻辑性和动力性
4. 将灵感分类表中的素材填充到幕方案表

## 产出

幕布局表（四部分内容 + 三关键事件）

## 检验问题

- 三个关键事件是否都确立？
- 每个事件是否不可逆且有推动力？
- 幕与幕之间的过渡是否自然？
