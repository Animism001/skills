# Phase 15: 写剧本

> 目标：从工作表到剧本正文

## 核心方法：三稿法

### 第一稿：素材的选择与排列

核心是围绕表达目标快速搭建素材，保持思维连贯。

- 用关键词和短句快速连接素材
- 明确每个故事单位的功能目标
- 补充环节工作表的环境描述
- 合理安排素材顺序（顺序影响情节逻辑、信息传递和节奏）
- **写什么比怎么写更重要**，不纠结细节

### 第二稿：描写与对白精修

#### 描写四原则

1. **可见**：只写视觉可观察内容
2. **周到**：说明到位、情绪到位、气氛到位
3. **简洁**：刚好表达清楚，不多不少
4. **准确**：不产生歧义，给后续工种留空间

#### 对白四大功能

1. **交流**：信息+情感的传递
2. **交代**：事件和人物背景的补充
3. **表现**：通过对话塑造人物
4. **独立审美**：对白本身的审美价值

#### 对白创作法

扮演人物，将人物的基因、经历、性格、能力、身份、欲望、动作全部代入自身。

### 第三稿：全盘检查与修改

#### 延时自检

完成第二稿后放置一段时间（至少2-3天），从创作情绪中冷静下来再理性检查。

#### 连续通读

集中时间一口气读完剧本，把握整体结构与节奏。

#### 他人反馈

请2名以上有故事感觉或文字能力的人通读，收集反馈。

## 换位思考验证（终检）

### 三层验证

**第一层：可理解性**
- 遮住脑补测试：只看文字本身，情节是否连贯？
- 陌生人测试：完全不了解背景的人能否理解？
- 时间线测试：读者能否理清事件顺序？

**第二层：情感共鸣**
- 触发事件：读者是否想继续看下去？
- 危机时刻：读者是否为主角捏一把汗？
- 高潮：读者是否感到情感被满足？

**第三层：注意力**
- 钩子测试：每一段落的结尾是否让读者想看下一段？
- 承诺测试：故事开头是否做出了某种承诺？结尾是否兑现了？
- 冗余测试：是否有段落删掉后不影响理解和情感？

## 检验问题

- 三稿是否都完成？
- 描写是否遵循四原则？
- 对白是否承担四大功能？
- 换位思考三层验证是否通过？
