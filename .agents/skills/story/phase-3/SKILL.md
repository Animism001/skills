# Phase 3: 七声部织体工作表

> 目标：构建故事的"总谱"，明确起点与终点

## 核心方法：七声部框架

故事不是单一线索，而是七个声部同时演奏的织体。

### 七个声部

| 声部 | 定义 | 检验问题 |
|------|------|----------|
| 1. 主题 | 故事要表达的核心理念或命题 | 能否用一个问句表达？ |
| 2. 人物 | 故事中的角色及其弧线 | 去掉这个人物，故事是否坍塌？ |
| 3. 事件 | 故事中发生的事情，按因果链排列 | 删除这个事件，后续事件是否还能发生？ |
| 4. 结构 | 事件的排列方式和节奏安排 | 当前结构是否最大化悬念和情感冲击？ |
| 5. 场景 | 故事发生的具体时空和氛围 | 这个场景能否用一句话概括其功能？ |
| 6. 对话 | 人物之间的言语交流 | 遮住角色名，能否仅凭说话方式区分角色？ |
| 7. 节奏 | 故事的快慢、松紧、轻重变化 | 画出情绪曲线，是否有起伏？是否有高峰？ |

## 操作流程

1. 将八要素重新整合为**七个声部**
2. 分别写出**开篇事件**和**高潮事件**时各声部的状态
3. 开篇事件需同时承担：背景交代、人物引出、体现负价值、足够精彩
4. 高潮事件需：集中解决核心问题、明确人物命运、表达主题
5. **拉开反差**：在人物线、事件线、主题线上设计起点与终点的状态距离，反差越大弧光空间越大

## 检验问题

- 七个声部是否都设计完整？
- 开篇与高潮的反差是否足够大？
- 每个声部的起点和终点是否清晰？
