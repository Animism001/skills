# 小说深度分析技能

这是一个专业的网文/小说分析工具，基于 D.E.S.I.R.E. Matrix 框架进行多维度的深度心理诊断和文风分析。

## 输入要求

用户提供小说/网文文本（可以是：
- 完整的章节内容
- 故事大纲或简介
- 剧情片段
- 多章节的连载内容）

## 输出格式

严格按以下四段式结构输出分析结果，每个模块都用代码块包裹：

---

# summary

```
[summary]
『第X日』
 - 1. [该日发生的事件1，填入事件与细节]
 - 2. [该日发生的事件2]

『第X日』
 - 1. [该日发生的事件1]
 - 2. [该日发生的事件2]

关键事件与叙事影响：
- 1. [重要事件概括]：[人物关系变化]
```

**summary 编写规则：**
- 按时间日期顺序记录
- 每条摘要控制在 20 字左右
- 使用主谓宾句式，精炼准确
- 关键事件需标注叙事影响（用 → 表示变化）
- 只记录剧情部分，角色设定不出现

# MATRIX

```
[MATRIX]
D - Drive: [核心驱动力]
E - Experience: [渴望体验的角色类型]
S - Shortcut: [思维捷径/认知偏误]
I - Identity: [身份原型/人设]
R - Risk: [风险偏好/预防焦点]
E² - Emotion: [情感偏好/冯内古特弧线]
```

**各维度分析指南：**

#### D - Drive（核心驱动力）
分析主角追求的核心需求：
- 胜任感：渴望展现能力、获得成就
- 归属感：渴望被接纳、建立联系
- 自主感：渴望独立自主、掌控命运
- 公平感：追求正义、打破不公
- 亲密感：渴望深度情感连接

#### E - Experience（体验角色）
分析主角渴望扮演的角色类型：
- 成就者：快节奏、行动导向
- 探索者：慢节奏、心理与环境描写
- 社交家：对话驱动、情感纠葛
- 反抗者：打破规则、逆袭反击
- 守护者：保护他人、承担责任

#### S - Shortcut（思维捷径）
分析主角的认知模式和决策方式：
- 容忍模糊 → 接受复杂情节
- 追求确定 → 偏好简单直接
- 社会认同 → 重视他人评价
- 权威原则 → 依赖规则和证据
- 互惠原则 → 重视等价交换

#### I - Identity（身份原型）
分析主角的身份认同：
- 智者：思辨、典雅的语言
- 颠覆者：直接、有冲击力
- 受伤者：敏感、防御性强
- 成长者：从弱到强的蜕变
- 复仇者：以其人之道还治其人之身

#### R - Risk（风险偏好）
分析主角的风险预防焦点：
- 预防焦点：极度规避损失、追求安全感
- 促进焦点：追求收益、敢于冒险
- 损失厌恶：对负面评价极度敏感
- 掌控需求：渴望微观、精细的描写

#### E² - Emotion（情感偏好）
分析主角渴望的情感弧线类型：
- 灰姑娘型/绝地反击型：压抑-爆发-反击-释然
- 英雄旅程型：冒险-成长-归来
- 悲剧型：希望-绝望-毁灭
- 治愈型：创伤-疗愈-和解
- 博弈型：智性交锋、悬念迭起

# WritingStyle

```
[WritingStyle]
[核心风格：根据 MATRIX 提炼的 3-5 个最关键风格词]

氛围与基调
- 基调: [根据 E² 填入：浪漫/悲剧/悬疑/紧张/热血/宁静/治愈/智性博弈等]
- 情感强度: [根据 R 和 D 填入：强烈而富有张力 / 内敛而克制 / 平和而舒缓]
- 类型元素: [根据 E²-Emotion 填入：戏剧冲突/哲学思辨/情感纠葛/奇观想象]
- 感官偏向: [根据 I 和 E² 填入]

节奏与步速
- 总体节奏: [根据 D-Drive 和 E¹ 填入：快节奏情节驱动 / 慢节奏氛围优先 / 张弛有度]
- 信息密度: [根据 I-Identity 填入：高密度信息量大 / 低密度留白较多]

叙事手法
- 叙事引擎: [根据 E¹ 和 S 填入：对话为核 / 描写为核 / 均衡驱动]
- 叙事焦点: [根据 E¹ 和 I 填入：聚焦内在心理 / 聚焦外在行动 / 聚焦环境氛围]

描写方式
- 描写层次: [根据 E² 填入：宏观微观交替 / 聚焦微观细节 / 聚焦宏观场面]
- 修辞偏好: [根据 E² 填入：多用象征暗喻 / 纯白描少用修辞 / 语言如刀精准直接]
```

**文风示例参考：**

**示例1：现实主义反击型**
```

- 基调: 现实伦理、紧张博弈
- 情感强度: 强烈而富有张力，内敛于行动中爆发
- 类型元素: 强烈的戏剧冲突、家庭伦理反击
- 感官偏向: 聚焦于人物微表情、动作细节与言语交锋

- 总体节奏: 快节奏，情节驱动，张弛有度
- 信息密度: 高密度，信息量大，无废话
叙事手法
- 叙事引擎: 对话为核，以高频、精炼且带刺的对话驱动情节
- 叙事焦点: 聚焦外在行动与逻辑推理
描写方式
- 描写层次: 聚焦微观细节（如手抖、眼神、语气变化）
```

- 奖励要与身份原型呼应

## 分析流程

1. **接收输入**：读取用户提供的小说文本
2. **提取 summary**：
   - 按时间顺序梳理事件
   - 提炼关键事件和叙事影响
   - 合并同类项，保持精简
3. **执行 MATRIX 分析**：
   - 逐维度分析主角心理
   - 注意维度间的相互影响
   - 提炼核心特征
4. **生成 WritingStyle**：
   - 基于 MATRIX 推导文风偏好
   - 保持语言精炼
   - 给出具体可操作的指导
5. **提出剧情建议**：
   - 规避区要具体明确
   - 推荐方向要符合角色逻辑
   - 起伏设计要有张力

## 注意事项

- 所有分析都要基于文本内容，客观中立
- summary 必须准确无误，不遗漏重要事件
- MATRIX 六个维度都要覆盖，不能缺失
- WritingStyle 要有可操作性，不能泛泛而谈
- 剧情建议要与前面分析自洽

## 输出示例

用户提供类似以下内容时：
> 秋云发现父亲把300万拆迁款全给大哥，父亲突发高血压住院。秋云带着微信记录去医院对峙，最终拿回200万。

输出应为：

# summary

```
[summary]
『第一日』
 - 1. 父亲将300万拆迁款全给大哥建平，秋云质问不公，父亲突发高血压住院。
 - 2. 医院内，大哥以回上海交首付为由推脱照顾责任。
 - 3. 秋云得知拆迁款按人头分配，她应得200万。

关键事件与叙事影响：
- 1. 拆迁款真相曝光：秋云得知被骗 → 亲情滤镜破碎
- 2. 微信记录反击：秋云夫妇绝地反击 → 夺回200万并确立边界
```

# MATRIX

```
[MATRIX]
D - Drive: 核心驱动力为"公平正义"与"自我保护"。追求在极度不公的原生家庭环境中打破道德绑架，夺回属于自己的合法权益。
E - Experience: 渴望体验"觉醒者"与"反击者"的角色。从默默付出的受害者，转变为逻辑清晰、手段果决的权益捍卫者。
S - Shortcut: 识别出反派利用"社会认同"和"道德绑架"进行操控。倾向于使用"权威原则"和"确凿证据"来打破认知偏误。
I - Identity: 构建"清醒、独立、不内耗的现代女性"及"家庭利益捍卫者"的身份认同，拒绝传统的"扶魔"或"愚孝"标签。
R - Risk: 预防焦点。极度规避"被道德绑架"、"强行大团圆/原谅"以及"自身财产和名誉的实质性受损"。
E² - Emotion: 偏好"压抑-爆发-反击-释然"的冯内古特弧线（灰姑娘型/绝地反击型）。追求逻辑严密的"打脸"爽感和现实主义的因果报应。
```

# 剧情建议

```
[剧情建议]
规避区
- 绝对规避"圣母心泛滥"或"强行原谅"的结局，主角不能对吸血亲戚心软
- 避免逻辑漏洞，反击必须基于合法合规的证据
- 避免主角在名誉或财产上遭受不可逆的实质性损失

风的核心驱动力
- 获得应有的公平与尊重，打破原生家庭的剥削链条
- 保护自己的小家庭，实现经济与人格的双重独立

推荐剧情方向
- 财产分割后的独立生活：投资事业，生活蒸蒸日上
- 反派自食恶果：大哥在上海的真实困境爆发

可设计的剧情起伏
- 父亲后续的养老危机：大哥拒不履行义务，父亲再次试图用苦肉计
- 舆论反转：村民看到大哥真面目后，舆论倒向秋云

剧情奖励
- 彻底摆脱原生家庭的吸血，获得心理上的完全自由
- 丈夫卫东的全力支持与夫妻感情的升温
- 看到曾经算计自己的人陷入困境（因果报应的爽感）
```
