# B站视频处理技能

## 功能概述

本技能提供以下功能：
1. **B站视频搜索** - 搜索指定关键词的B站视频
2. **UP主视频获取** - 获取指定UP主的所有视频链接
3. **视频字幕转换** - 将B站视频转换为字幕文件

## 使用场景

- 用户需要搜索B站上的特定视频
- 用户需要获取某个UP主的所有视频链接
- 用户需要将B站视频转换为字幕文件

## 操作流程

### 1. B站视频搜索

**输入格式**：
```
搜索B站视频 "关键词"
```

**示例**：
```
搜索B站视频 "AI写作教程"
```

**执行步骤**：
1. 调用B站搜索API搜索指定关键词
2. 返回搜索结果，包含视频标题、链接、UP主、播放量等信息

### 2. UP主视频获取

**输入格式**：
```
获取B站UP主 "UP主名称" 的视频
```

**示例**：
```
获取B站UP主 "星月凌云" 的视频
```

**执行步骤**：
1. 搜索并定位指定UP主
2. 获取UP主的所有视频列表
3. 保存视频链接到文件
4. 返回视频数量和保存路径

### 3. 视频字幕转换

**输入格式**：
```
转换B站视频字幕 "视频链接" 到 "输出目录"
```

**示例**：
```
转换B站视频字幕 "https://www.bilibili.com/video/BV1zK4y1N7hF" 到 "subtitles"
```

**执行步骤**：
1. 验证视频链接
2. 下载视频字幕（优先使用CC字幕，其次AI字幕，最后使用Whisper转录）
3. 保存字幕文件到指定目录
4. 返回字幕文件路径

## 核心脚本

### 搜索脚本

使用 `scripts/search_bilibili.py` 进行B站视频搜索。

### UP主视频获取脚本

使用 `scripts/get_up_videos.py` 获取UP主的所有视频。

### 字幕转换脚本

使用 `scripts/bilibili_transcript.py` 转换视频为字幕。

## 注意事项

1. 搜索结果可能会受到B站API限制，可能需要分页获取
2. UP主视频获取可能需要处理大量视频，执行时间较长
3. 字幕转换依赖于视频是否有可用字幕，无字幕的视频会使用Whisper转录
4. 所有操作都需要遵守B站的使用条款和API限制
