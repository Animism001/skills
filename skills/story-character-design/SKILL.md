# 人物设计五步法

好人物不是从外貌描写开始的，而是从"力向"开始的——这个人物在故事中施加什么力、承受什么力、力的方向如何变化。先定力向，再设属性，人物才能活起来。

## 核心原则

**力向优先**：人物的一切属性（性格、背景、外貌）都服务于TA在故事中的力学功能。先确定人物在故事力学中的位置，再填充细节，这样每个细节都有存在的理由。

## 五步操作

### Step 1：确定人物的力学功能

问三个问题：

1. **这个人物对主角施加什么力？**（推动/阻碍）
2. **这个力来自哪里？**（利益/情感/价值）
3. **这个力在故事中如何变化？**（力的方向是否翻转）

**输出**：一句话描述人物的力学功能。
- 示例："在故事前半段推动主角成长，后半段因利益冲突转为阻力"
- 示例："始终是主角的阻力，但阻力来源从情感转为价值"

### Step 2：定义"想要"和"需要"

每个人物都有自己的"想要"和"需要"——不仅是主角。

- **想要**：这个人物在故事中追求的外在目标
- **需要**：这个人物真正需要的内在成长
- **张力**："想要"和"需要"之间的矛盾

**关键**：配角的"想要"和"需要"必须与主角的产生交叉——
- 配角的"想要"可能阻碍主角的"想要"（冲突）
- 配角的"需要"可能帮助主角的"需要"（共鸣）
- 配角的"想要"可能帮助主角的"想要"但阻碍双方的"需要"（复杂联盟）

**输出**：
```
人物：[名字]
想要：[外在目标]
需要：[内在成长]
张力：[想要和需要的矛盾]
与主角的交叉：[如何与主角产生互动]
```

### Step 3：设计人物弧线

人物弧线 = 从故事开始到结束的内在变化轨迹。

**四种基本弧线**：

| 弧线类型 | 起点 | 终点 | 适用 |
|----------|------|------|------|
| 成长型 | 有缺陷 | 克服缺陷 | 主角最常见 |
| 堕落型 | 有优点 | 被缺陷吞噬 | 悲剧主角/反派 |
| 平坦型 | 不变 | 不变 | 配角/象征性角色 |
| 翻转型 | 好变坏/坏变好 | 与起点相反 | 最具戏剧性 |

**弧线设计要点**：
- 弧线必须有触发点——什么事件启动了变化？
- 弧线必须有阻力——人物不会轻易改变
- 弧线必须有代价——改变意味着失去什么

**输出**：一段弧线描述，包含起点、触发点、阻力、转折、终点。

### Step 4：填充人物属性

现在才到了填充细节的时候。每个属性都必须回答一个问题：**这个属性如何支撑人物的力学功能和弧线？**

| 属性类别 | 填充内容 | 必须回答的问题 |
|----------|----------|----------------|
| 性格特征 | 2-3个核心特质 | 这个特质如何导致TA的"想要"或阻碍TA的"需要"？ |
| 背景故事 | 关键经历 | 什么经历塑造了TA现在的力向？ |
| 外貌特征 | 1-2个标志性特征 | 这个特征如何暗示TA的内在？ |
| 说话方式 | 语言风格 | TA的说话方式如何揭示性格和隐藏的想法？ |
| 习惯/怪癖 | 1个独特习惯 | 这个习惯如何与TA的"需要"相关？ |

**原则**：每个属性都是冰山的一角，暗示水面下的深度。没有"无用"的细节——如果删掉这个属性不影响对人物的理解，就不需要。

### Step 5：验证人物的功能性

用三个检验确认人物设计完成：

1. **删除测试**：删掉这个人物，故事是否坍塌？如果不坍塌，这个人物没有存在必要
2. **替换测试**：用另一个完全不同的人物替换，故事是否依然成立？如果成立，原人物不够独特
3. **弧线测试**：人物的起点和终点是否形成了有意义的变化？如果没有变化或变化无意义，弧线需要重做

## 次要人物三原则

次要人物不需要完整的五步设计，但必须遵循三个原则：

1. **功能化**：次要人物的存在只为完成一个明确功能（提供信息/制造障碍/代表某种观点）
2. **精简化**：只保留支撑功能的属性，不浪费笔墨
3. **差异化**：即使功能相似，也要有区分度——读者不应混淆两个次要人物
