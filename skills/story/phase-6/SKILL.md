# Phase 6: 幕的织体

> 目标：在幕层面理顺逻辑、调优动力

## 核心方法：一句话序列故事检验

### 检验公式

```
[谁] 为了 [什么目标]，面对 [什么障碍]，做了 [什么行动]，结果 [什么变化]。
```

### 幕内逻辑编织

将要素连成因果链：
```
身份 → 欲望 → 动作 → 障碍 → 结果 → 意义
```

用"一句话幕故事"检验通顺。

### 幕内动力编织

调节四参数：
- 身份难易
- 欲望强弱
- 动作正反
- 障碍大小

### 幕间编织

- **逻辑编织**：以关键事件为节点验证要素跨幕演变合理性
- **动力编织**：通过调整欲望与结果间**鸿沟**控制稳定性，鸿沟越大动力越强

## 操作流程

1. 幕内逻辑编织：用一句话幕故事检验因果链
2. 幕内动力编织：调控四参数
3. 幕间逻辑编织：验证跨幕演变
4. 幕间动力编织：调控鸿沟

## 优先原则

- 表达目标 ＞ 逻辑性 ＞ 动力性
- 已知主导，亮点优先

## 产出

幕织体工作表（含"一句话幕故事"）

## 检验问题

- 每个幕的因果链是否通顺？
- 动力是否足够（鸿沟是否足够大）？
- 幕与幕之间的衔接是否自然？
