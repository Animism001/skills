# Phase 11: 序列织体

> 目标：在序列层面深化逻辑与动力

## 核心方法

### 序列定义

10个路标事件划分10个序列，每个序列是故事的一个有机组成部分。

### 序列内编织

1. 理顺因果：确保序列内事件有清晰的因果关系
2. 调控四参数：
   - 身份难易
   - 欲望强弱
   - 动作正反
   - 障碍大小

### 序列间编织

1. 确保路标事件衔接合理
2. 用鸿沟法控制稳定性：鸿沟越大，动力越强

### 一句话序列检验

每个序列写"一句话序列故事"：
```
[谁] 为了 [什么目标]，面对 [什么障碍]，做了 [什么行动]，结果 [什么变化]。
```

## 操作流程

1. 10个路标事件划分10个序列
2. 序列内编织：理顺因果+调控四参数
3. 序列间编织：确保衔接合理+鸿沟法
4. 每个序列写"一句话序列故事"检验逻辑
5. 遵循优先顺序：表达目标 → 逻辑性 → 动力性

## 检验问题

- 每个序列的因果链是否通顺？
- 序列间的衔接是否自然？
- 序列之间的鸿沟是否足够大（动力是否足够）？
