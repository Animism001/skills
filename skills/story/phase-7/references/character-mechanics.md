# 人物关系力学分类详解

人物关系不是静态的标签，而是动态的力。

## 核心模型

人物关系分为两个维度：

### 维度一：处事关系（行动层面）

| 关系类型 | 定义 | 效果 |
|----------|------|------|
| 动力关系 | 一方推动另一方前进 | 加速故事推进 |
| 阻力关系 | 一方阻碍另一方前进 | 制造冲突和悬念 |

### 维度二：相互关系（情感层面）

| 关系类型 | 定义 | 张力来源 |
|----------|------|----------|
| 利益关系 | 因共同或对立的利益而联结 | 利益冲突、脆弱联盟 |
| 情感关系 | 因爱、恨、愧疚等情感而联结 | 情感与理性的冲突 |
| 价值关系 | 因信念、价值观的相同或对立而联结 | 价值观碰撞、信念考验 |

## 关系矩阵

```
人物关系矩阵

        | 处事关系  | 相互关系
--------|-----------|----------
A对B    | 动力/阻力 | 利益/情感/价值
B对A    | 动力/阻力 | 利益/情感/价值
```

## 关系不对称性

关系的不对称性是最强的戏剧张力来源：

- A把B当动力，B却对A形成阻力（暗中背叛）
- A与B是利益关系，但B对A产生了情感关系
- A对B是价值认同，B对A只是利益利用

## 关系弧线

人物关系也有弧线：

| 弧线类型 | 描述 | 戏剧效果 |
|----------|------|----------|
| 动力→阻力 | 盟友变敌人 | 背叛的冲击 |
| 阻力→动力 | 敌人变盟友 | 联盟的惊喜 |
| 利益→情感 | 交易变真情 | 真情的升华 |
| 情感→价值 | 私情变信念 | 信念的坚定 |

## 处事关系设计原则

### 层次

通过欲望强弱与能力大小差异，使对抗力有层次：
- 主角弱/对手强 → 成长空间大
- 主角强/对手弱 → 胜利更珍贵
- 双方接近 → 最激烈的对抗

### 平衡

对抗双方旗鼓相当，故事才精彩：
- 一方太弱 → 没有张力
- 一方太强 → 没有希望

### 变化

力向可以转变：
- **叛徒**：原本是动力，后来变成阻力
- **投诚者**：原本是阻力，后来变成动力
- **双面间谍**：同时对不同人发挥不同力向

## 关系筹码

关系筹码越大，关系的戏剧张力越大：
- 杀父之仇 vs 微博骂战
- 至死不渝的爱 vs 逢场作戏

## 操作流程

1. 列出所有人物对
2. 为每个人物对填写关系矩阵
3. 寻找关系的不对称和翻转点
4. 设计关系弧线
