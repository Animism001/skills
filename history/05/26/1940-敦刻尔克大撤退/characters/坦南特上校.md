# 威廉·坦南特上校 — 人物设定卡片

## 基本信息
- **全名**：Sir William George "Bill" Tennant
- **当时身份**：英国皇家海军上校
- **年龄**：49岁
- **关键角色**：敦刻尔克撤退战术监督，"海滩主管"（Beachmaster）

---

## 外在画像

```
体态：中等偏高，精干，像一把磨了多年的刀——不显眼但随时可以出鞘
面容：方脸，下巴结实，颧骨高耸，皮肤被海风和硝烟熏成暗红色
眼睛：深褐色，目光锐利而快速——像在人群中寻找登船口的人
穿着：卡其色海军制服，领口敞开，袖子卷到肘部——他不是来视察的，是来干活的
声音：清晰、简短、有力——每个词都像钉子，不多不少钉在要点上
习惯动作：用手指点数——数人、数船、数时间；站在高处俯瞰海滩，双手叉腰
气味：海水的咸腥、柴油的刺鼻、沙滩上被太阳暴晒的腐物、防波堤木板的霉味
```

## 内在驱动

```
核心信念：在最混乱的地方，最简单的方法往往最有效——别想太复杂，找到一条路，让人走过去
深层恐惧：不是怕死，而是怕自己没找到那条路——如果有人因为他的疏漏而没能登船
性格矛盾：他是最冷静的人，却身处最混乱的战场；他是最谦逊的人，却做出了最大胆的决定
内在冲突：启用防波堤是天才的直觉还是走投无路的赌博？他自己可能也不确定
隐藏动力：他不是英雄主义者，而是一个务实到骨子里的人——问题在那里，他必须解决它
```

## 人物弧线

```
起点（5月27日）：被派往敦刻尔克。他坐船穿越德军轰炸抵达海滩，看到的第一眼是——40万人在沙滩上等待死亡，港口被炸瘫痪，原定方案完全失效。

中点（5月27日傍晚）：他看到了东防波堤。一条1,200-1,400码长的简易木制防波堤，宽仅容8人并排行进，千疮百孔，人满为患。所有人都觉得它不可能当码头用。他走上去，木板在脚下嘎吱作响。他做了一个决定。

转折（5月28-31日）：5天内约20万人通过东防波堤登船。一条从未被设计为码头的防波堤，成了敦刻尔克的"生命线"。

终点（6月2-3日）：英军撤退基本完成。坦南特作为最后一批撤离者之一离开海滩。他回头看了一眼——空荡荡的防波堤在海浪中摇晃，像一条疲惫的旧手臂。

弧线本质：从"在混乱中寻找秩序"到"用创造力逆转死局"——坦南特证明了一件事：在最危急的时刻，最重要的不是权力，而是创造力。
```

## 关键场景预设

### 场景1：5月27日抵达敦刻尔克
- **地点**：敦刻尔克海滩
- **情境**：坦南特坐船穿越轰炸抵达，发现港口瘫痪
- **感官锚点**：脚踩进沙滩的第一步——沙是湿的、冷的，混着柴油和血的味道；头顶有斯图卡俯冲轰炸机的尖啸
- **核心动作**：坦南特站在沙滩上，环顾四周——数万人蹲在沙滩上，像退潮后的海豹
- **情感层次**：不是恐惧，而是紧迫——每一分钟都有人在死去，他必须立刻找到办法

### 场景2：发现并启用东防波堤
- **地点**：敦刻尔克东防波堤
- **情境**：港口瘫痪，从海滩登船效率极低，坦南特需要找到替代方案
- **感官锚点**：防波堤的木板在脚下嘎吱作响，海水从缝隙中涌上来，浸湿了他的靴子
- **核心动作**：坦南特走完防波堤的全长，站到尽头，回头看——1,200码的木板路，通向深水区。他做了一个手势：从这里上船
- **情感层次**：没有灵光乍现的戏剧性——只有一个务实者的判断：这条路能走，那就走

### 场景3：最后离开海滩
- **地点**：敦刻尔克东防波堤
- **情境**：英军撤退基本完成，坦南特作为最后一批撤离者
- **感官锚点**：空荡荡的防波堤，木板在海浪中起伏；远处有零星的枪声——法军后卫还在战斗
- **核心动作**：坦南特回头看了一眼防波堤，然后转身登船
- **情感层次**：没有胜利的喜悦——他救了20万人，但9万人没能走掉，法军还在身后战斗
