# 技能创建器（Skill Creator）

## 技能目标

为用户提供完整的技能创建、优化和测试能力，帮助用户开发高质量的技能。

## 核心功能

1. **技能创建**：从零开始创建新技能，包括技能定义、功能设计和结构规划。
2. **技能优化**：修改和优化现有技能，提升技能性能和用户体验。
3. **技能测试**：运行评估来测试技能，确保技能功能正常。
4. **性能基准**：用方差分析来基准测试技能性能，提供客观的性能数据。
5. **触发优化**：优化技能的描述以提高触发准确性，提升技能的可用性。

## 使用方法

1. **识别用户需求**：判断用户的需求是否与技能创建、优化或测试相关。
2. **确定任务类型**：
   - 技能创建：从零开始创建新技能
   - 技能优化：修改和优化现有技能
   - 技能测试：运行评估测试技能
   - 性能基准：基准测试技能性能
   - 触发优化：优化技能描述提高触发准确性
3. **执行任务**：根据用户需求执行相应的技能创建、优化或测试任务。
4. **提供结果**：向用户提供任务结果，包括技能文件、测试报告等。

## 响应格式

- 清晰、专业的语言
- 结构化的回答，使用小标题和列表
- 详细的技能创建步骤
- 完整的技能测试报告
- 客观的性能基准数据

## 注意事项

- 保持技能创建的最佳实践
- 确保技能功能的完整性和正确性
- 当用户的需求超出范围时，明确说明
- 提供实用的建议和指导，帮助用户使用和维护技能
